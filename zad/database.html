❌ الجداول المطلوبة (غير موجودة)

جداول يجب إنشاؤها لتشغيل محرك المزادات

🚗 vehicles — جدول السيارات

id — PK
make (الماركة: تويوتا، BMW...)
model (الموديل: كامري، X5...)
year (سنة الصنع)
color, mileage, vin_number
condition: new/used/damaged
engine_size, transmission, fuel_type
inspection_report (تقرير الفحص)
ownership_documents (وثائق الملكية)
city, region, country
created_by FK→users
status: pending/approved/live/sold

🖼️ vehicle_images — صور السيارات

id — PK
vehicle_id FK→vehicles
image_path (مسار الصورة)
is_primary (الصورة الرئيسية)
sort_order (ترتيب العرض)
caption (وصف الصورة)

🔨 auctions — جدول المزادات

id — PK
vehicle_id FK→vehicles
title_ar, title_en (عنوان المزاد)
start_price (السعر الابتدائي)
reserve_price (سعر التحفظ)
min_bid_increment (أدنى زيادة)
deposit_amount (مبلغ التأمين)
starts_at, ends_at (وقت البدء والانتهاء)
status: scheduled/live/ended/cancelled
current_price (آخر سعر)
winner_id FK→users
created_by FK→users

💬 bids — جدول المزايدات

id — PK
auction_id FK→auctions
user_id FK→users
amount (مبلغ المزايدة)
is_winning (هل هو الفائز الحالي)
ip_address (للتحقق من التلاعب)
created_at (وقت المزايدة بالثانية)

🔒 auction_deposits — تأمينات المزادات

id — PK
auction_id FK→auctions
user_id FK→users
wallet_id FK→wallets
amount (مبلغ التأمين المحجوز)
status: held/released/forfeited
released_at (وقت الإفراج)

⭐ watchlists — قائمة المراقبة

id — PK
user_id FK→users
auction_id FK→auctions
created_at (وقت الإضافة)

🔔 notifications — الإشعارات

id — UUID
type (نوع الإشعار)
notifiable_id, notifiable_type
data (JSON: الرسالة والرابط)
read_at (وقت القراءة)

💳 payments — المدفوعات

id — PK
user_id FK→users
auction_id FK→auctions
amount, currency
payment_gateway (moyasar/hyperpay)
transaction_id (معرف البوابة)
status: pending/paid/failed/refunded
