🏗️ بنية التطبيق المقترحة

كيفية تنظيم الكود للمراحل القادمة

🌐

طلب المستخدم

المتصفح يرسل HTTP Request إلى Laravel Router الذي يوجهه للـ Middleware والـ Controller

⚙️

طبقة الخدمات

Services Layer: WalletService, AuctionService, BidService — منطق الأعمال مفصول عن الـ Controller

🗄️

طبقة البيانات

Eloquent Models مع Relationships محكمة وScopes للاستعلامات الشائعة

📡

البث اللحظي

Events → Broadcasting → Laravel Echo → WebSocket Channel يُحدّث واجهة المزايد فوراً

⏰

المهام الخلفية

Queue Jobs للإشعارات والإيميلات. Scheduled Commands لإنهاء المزادات تلقائياً

🔔

الإشعارات

Laravel Notifications → Database + Mail + Broadcast للتبليغ الفوري والمحفوظ

💻 أوامر التثبيت المطلوبة

الأوامر الجاهزة للنسخ واللصق لتثبيت الحزم

# حزم PHP المطلوبة
composer require laravel/reverb intervention/image-laravel
composer require maatwebsite/excel barryvdh/laravel-dompdf
composer require pusher/pusher-php-server laravel/horizon
 
# حزم JavaScript المطلوبة
yarn add laravel-echo pusher-js chart.js flatpickr alpinejs
yarn add sortablejs glightbox
 
# إعداد Laravel Reverb (WebSockets)
php artisan reverb:install
php artisan reverb:start
 
# Migrations الجديدة المطلوبة
php artisan make:migration create_vehicles_table
php artisan make:migration create_vehicle_images_table
php artisan make:migration create_auctions_table
php artisan make:migration create_bids_table
php artisan make:migration create_auction_deposits_table
php artisan make:migration create_watchlists_table
 
# Models وServices المطلوبة
php artisan make:model Vehicle -mf
php artisan make:model Auction -mf
php artisan make:model Bid -mf
php artisan make:service AuctionService
php artisan make:service BidService
 
# Events للبث اللحظي
php artisan make:event BidPlaced
php artisan make:event AuctionEnded
php artisan make:notification AuctionWonNotification
php artisan make:notification OutBidNotification
